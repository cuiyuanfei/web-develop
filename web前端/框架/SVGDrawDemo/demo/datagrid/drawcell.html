员工帐号

#

Description

  1. 开发者可以定义列Column的renderer函数，或监听Grid的drawcell事件，来自定义显示单元格内容。
  2. 单元格内容是一个html字符串，可以是：文本、图标、超链接、按钮等任意HTML元素。
  3. 本例显示如下类型格式化处理：
    1. String：员工帐号，设置背景色
    2. Boolean：性别，用图片标识男、女。
    3. Number：薪资，超过10000，红色高亮显示。
    4. Date：出生日期，格式化日期字符串显示。
    5. Array：学历，通过id显示文本。
    6. 最后一列显示超链接。
